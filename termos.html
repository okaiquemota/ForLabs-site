1. Aceitação dos Termos

Ao acessar e utilizar o sistema ForLabs (o "SaaS"), você concorda em cumprir e ficar vinculado a estes Termos de Uso. Caso não concorde com qualquer parte destes termos, você não deverá utilizar a plataforma.

2. Descrição do Serviço

O ForLabs é uma plataforma de gestão e controle de qualidade laboratorial oferecida como Software as a Service (SaaS). A ferramenta permite o cadastro de testes dinâmicos, análises de pigmentos, gestão de manuais e emissão de laudos. O sistema atua de forma isolada para cada empresa cliente (multi-tenant).

3. Contas e Segurança

  • É responsabilidade da empresa cliente manter a confidencialidade de suas credenciais de acesso e das contas de seus colaboradores (Analistas, Administradores, etc.).
  • Todas as atividades realizadas sob uma conta de usuário são de total responsabilidade da empresa titular. O ForLabs registra trilhas de auditoria para garantir o controle interno do cliente.

4. Propriedade dos Dados

Os dados inseridos no ForLabs (resultados de laudos, fotos de amostras, PDFs de manuais, dados de pigmentos) pertencem integralmente a você e à sua empresa. Nós apenas fornecemos a infraestrutura tecnológica para armazenamento e processamento desses dados, garantindo a sua confidencialidade.

5. Disponibilidade e Backups

Trabalhamos para garantir que a plataforma tenha alta disponibilidade (uptime). O sistema permite que os Administradores de cada empresa façam o download (backup) da base de dados bruta em arquivos CSV ou JSON a qualquer momento pelo painel de controle.
